# Hyprland Dotfiles

A **feature-rich Hyprland** dotfiles setup focused on productivity, aesthetics, and a smooth Wayland workflow.

> ⚠️ These dotfiles reflect my personal setup and may overwrite existing configs.  
> **Back up your files before installing.**

---
 

## Requirements

Before installing, make sure you have:

- Hyprland
- Wayland session
- `git`, `bash`, core GNU utilities

> Additional dependencies are handled by the install script.

---

## Installation

Clone the repository and run the installer:

```bash
git clone https://github.com/anmolpaweriya/hyprdots
cd hyprdots
./Scripts/install.sh
